Generating Plots

Two types of plot are available from the "Detailed Grade Report". Curve fits will be displayed if four or more points exist. Non heat treated data is displayed in red, heat treated in blue. Curve fit equations are displayed.
1. Plot property against density
Click the graph icon or adjacent text link on the required property shown in the table. This produces a plot of the property against density.



Rescale Plot / Select Curve Fit / Back to grade report
To rescale the axes, scroll to the bottom of the plot and enter the max and min values required. Select the curve fit required from the drop down menu. Then Click Accept. Select "Back to grade report" to plot another property.



Save Plot to File
Place mouse pointer over grey plot background until save icon bar appears. Then, Right Mouse Click to get save options. Use Save Picture As... to name and save the plot to a file.
2. Cross Plot
To plot one property against another, click the Cross Plot link.



Select which properties are required for X and Y axes from the drop down menus. Then click OK, to display plot.
