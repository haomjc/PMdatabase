Advanced Search and Download of Data to a Report

In this example the advanced search is used to find a material grade with both mechanical and fatigue endurance data. A report is created allowing a visual comparison to be made how Fatigue Endurance Limit varies with UTS. This is achieved by putting the plots of each property versus density side by side.

Select "Property Search" from the Advanced Search Tab.



Select material type "Ferrous" and click OK.



Select Materials with "No" Heat Treatment and click OK.



Click tick box for "UTS" followed by tick box for "Axial Fatigue Endurance Limit 50% certainty of survival". Scroll to the bottom of the page and click OK. This will find material grades with the selected Mechanical "OR" Fatigue properties selected.



Enter UTS greater than 600MPa & Axial Fatigue Endurance Limit 50% certainty of survival, greater than 150MPa. Click Accept.



Review the search criteria, save the search, then do the search. The edit link can be selected here to make any changes required.



When the search is complete click on the top result of the grade results list to go to the Detailed Grade Report for that grade.



When "Detailed Grade Report" is displayed, scroll to bottom of page and use the Excel Download button to save the grade details to a CSV file. Open this file in Excel by double clicking it from Windows Explorer. Once opened, reformat the tabulated data as required and save as an Excel Workbook File.



From the "Detailed Grade Report", create plots of "UTS v Density" and "Axial Fatigue Endurance v Density". Re-scale plots to allow visual comparison and save to file. Use "Help on Plots" to learn how to do this. Use the back to Grade Report after each plot.

Create the technical report in MS Office using the downloaded Excel data and plot files. Use Copy/Paste-Special, Picture (Windows Metafile) to transfer data from Excel into Word or Powerpoint. Use Insert, Picture from file, to add the plots.

The following report can now be used as a material record and to make an engineering judgement on how Axial Fatigue Endurance will vary with UTS.
