System help

This is general help. For help on specific functions within the powder metal database click the context sensitive help icon in the frame header of the function .

Navigation

Many of the search capabilities are provided as 'wizards' in which a series of screens lead you to the result. If you need to navigate back from a screen and in the absense of navigation in the browser display area, you can use the browser's 'back' button.

Update Account Details

This option allows you to change your name, company, address, e-mail, telephone number, subscription to the mailing list and password. You must select the 'Change Password' checkbox in order to enter a new password. The system will then change your details.

Units calculator

The units calculator enables you to easily convert values between a wide range of scientific and engineering units.

Print Page

Prints the currently displayed page.

View Data Cart

The data cart can be used to save the results of one or more searches. It uses the common 'shopping cart' analogy to save the results which were added to the cart. The View Data Cart option displays the current list of saved grades. Detailed information for each grade saved on the cart can be accessed by clicking the hyperlink of the grade name.

Empty Data Cart

Clears the data cart of any grades previously saved.

More Information

When you see this button there is more detailed technical background available.

Output formats

CSV files

Comma separated values - some search results can be exported or saved as CSV files. This is a text based format that canbe imported by popular database and spreadsheet software such as MS Excel.

PDF files

Some advanced searches can be saved as Adobe Acrobat PDF files instead of being displayed on screen. You will need the free Acrobat Reader to view the results. (see www.adobe.com)

FEA files

Some search results can be exported as FE file which are FE files that can be in either NASTRAN, ABAQUS or ANSYS format.

Show\Close shortcuts

The Powder Metal Database has MS Outlook style menu bars that you can click to reveal icons that will run its processes (note that Outlook style menus are only one level deep). You can hide the menu bars with the Close shortcuts button. This will make the browser display area full screen.
